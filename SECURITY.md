# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Argus itself, please do NOT open
a public issue. Instead, report it privately by emailing the maintainer via
the contact information on the GitHub profile: https://github.com/kentunji

You can expect an initial response within 7 days.

## Responsible Use

Argus is a security testing tool. You must only use it on systems you own
or have explicit written permission to test. Unauthorized use against systems
you do not own or have permission to test is illegal in most jurisdictions
and is not the intended purpose of this project.

The authors accept no liability for misuse.
